# PC・スマホ実践アプリ

「分からなくても、画面を観察し、小さく試し、必要なら戻して自分で進める」力を育てる、デジタル初心者向けの体験型Web教材です。

Windows、Mac、iPhone、Androidを模した安全な練習環境で、実データや外部サービスへ影響を与えずに操作できます。アカウント登録、外部API、実際の送信・発信・撮影は行いません。

## 学習設計

- 4つの利用環境：Windows、Mac、iPhone、Android
- 8エリア × 4ミッション＝全32ミッション
- 7つの横断能力：観察、操作、文字、情報、管理、安全、問題解決
- 前半は「見本あり」、中盤は「手掛かり中心」、後半は「自力チャレンジ」
- 操作順ではなく、最終状態と学習証拠で完了を判定
- ヒントは方向 → 機能名 → 環境別の具体操作の3段階
- 誤操作を減点せず、安全にやり直した過程も記録

8つの学習エリアは次のとおりです。

1. 触る・動かす
2. 画面とアプリを行き来する
3. 文字を扱う
4. 検索して確かめる
5. ファイルと写真を扱う
6. 安全に進める
7. 困ったときに解決する
8. 自力で組み合わせて解く

## 実践環境

- クリック、ダブルクリック、右クリック、長押し、スクロール、ドラッグ
- アプリの起動・終了・切替、戻る、メニュー発見
- 文字入力、範囲選択、コピー・貼り付け、編集の取り消し
- 検索語の改善、タブ比較、発信元・更新日の確認
- ダウンロード、保存場所、名前変更、移動、添付前レビュー
- 権限判断、アカウント復旧、確認画面、怪しいメッセージ
- エラー読解、Wi-Fi切り分け、公式ヘルプ検索、代替手段
- 電話・カメラを含む、スマートフォン向けの安全な自由練習

最終4ミッションでは、答えの手順ボタンを表示せず、検索欄、ファイル名・保存場所、設定内検索とスライダー、原因選択と権限設定を実際に操作して解決します。

## 主な画面

- `/start`：環境と操作方法の選択
- `/journey/[environment]`：32ミッションの学習地図
- `/mission/[environment]/[missionId]`：目的確認 → 準備 → 実践 → 証拠確認
- `/practice/[environment]`：課題なしの自由練習
- `/progress`：完了数、能力別の証拠、ヒント・復旧状況
- `/toolbox`：困ったときの確認方法
- `/settings`：文字サイズ、コントラスト、動きの軽減、記録削除

旧URLの14ステージは、内容に対応する新ミッションへ転送します。

## 技術構成

- Next.js 16（App Router）
- React 19
- TypeScript
- CSS（外部UIライブラリ不使用）
- Local Storage（進捗・表示設定の端末内保存）

主要な実装は以下です。

```text
src/
├─ app/                      # 画面とルート
├─ content/journey.ts        # 8エリア・32ミッションの定義
├─ features/journey/         # 開始、地図、エリア、ミッション、自由練習
├─ features/lab/             # 8種類の実践ワークスペースと状態判定
├─ features/progress/        # 学習証拠・進捗・設定の保存
└─ lib/journey-types.ts      # カリキュラムと成功条件の型
```

調査根拠、現行版の課題、カリキュラム、環境別操作、UI/UX、実装方針の詳細は [`docs/RESEARCH_AND_REDESIGN.md`](docs/RESEARCH_AND_REDESIGN.md) にまとめています。

## ローカル実行と検証

Node.js 20.9以降で実行します。

```bash
npm install
npm run dev
```

[http://localhost:3000](http://localhost:3000) を開いてください。

```bash
npm run lint
npx tsc --noEmit
npm run build
```

本番ビルドでは、4環境 × 32ミッションを含む全214ページを生成します。公開前にはデスクトップ・スマートフォン幅、キーボード操作、主要ミッションの完了、進捗保存、JavaScriptエラーを実ブラウザで確認します。

## 保存とプライバシー

氏名やメールアドレスは保存しません。完了ミッション、能力の証拠、ヒント段階、復旧回数、自由練習、表示設定だけを現在のブラウザ内へ保存します。別端末への同期は行わず、設定画面から削除できます。
